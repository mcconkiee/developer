Detailed Description

PoyntOrderItemObject order items are essential for the makeup of a PoyntPaymentObject. All order items declare their amount (in cents) and other details attached to accurately describe payments

PoyntOrderItemObject *item = [[PoyntOrderItemObject alloc] initWithSku:"unique.sku.name" unitPrice:200 quantity:2.1];
